Conditional Design Assignment

First Attempt
  1. Each person should choose the color (red, black, and blue), decide where to start and draw arrows in clockwise direction.
  2. The new arrow can only start at the corner of the previous person’s arrow and it should be bent once.
  3. Only red arrows could intersect with other arrows.
  4. Draw the red dots for each intersection.
  5. Stop when there is no room for arrows anymore.

Second Attempt
  1. Each person should choose the color (red, black, and blue), decide where to start and draw arrows in clockwise direction.
  2. The new arrow can only start at the corner of the previous person’s color and it should be bent once.
    (If person before you have red, your arrow can only be drawn from that color.)
  3. Only red arrows could intersect with other arrows.
  4. Draw the red dots for each intersection.
  5. Stop when there is no room for arrows anymore.
